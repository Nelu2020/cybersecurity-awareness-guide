# What I learned

Password Security

How strong passwords such as passwords length, complexity, expiration and history help protect accounts from unauthorized access. A strong password should be at least 12 characters long. Types of numbers and characters must be included, force users to change their passwords periodically, and prevent users from using old passwords are important.

Use of Multi-factor Authentication

Why multi-factor authentication (MFA) is crucial and important. MFA helps secure systems by requiring more one than method of verification (Type 1, Type 2, Type 3).

Phishing attack

How people get trick by phishing emails. This attack refers to trick users into giving personnal information. To stay safe, do not click unkown links, do not open suspicious attachments and verify the email's sender. These are some basics safe habits.

Why using public Wi-Fi can be risky

Public Wi-Fi networks are often unsafe and logging to sensitive accounts when using them can harm you.
